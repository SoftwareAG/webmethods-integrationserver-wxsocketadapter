webMethods Sample Adapter Listener Configuration Online Help

This is a sample online help page for the Sample Adapter's Listener Configuration page in the Integration Server Administrator.

You reach this page when you click one of the following:

  • Listeners > Configure New Listener > Sample Server Listener > Help

  • Listeners > View > Help

  • Listeners > View > View Notification Order > Help

  • Listeners > Copy > Help

  • Listeners > Edit > Help

  • Listeners > Edit Notification Order > Help
